virtual bool sxsdk::exporter_interface::can_export_to_stream ( bool &  uses_filesavedialog  )  [inline, virtual]

オーバライド可。プラグインがShadeから渡されたストリームにデータを書き出すことができる場合には、trueを返す。プラグインで独自にファイルを開くなど、ストリームを渡される必要がない場合には、オーバライドしてfalseを返す。

オーバライドしないかtrueを返すと、Shadeがファイルを開くにより排他制御が行われ、stream_interface 以外からの書き込みがブロックされる。プラグインが独自にファイルを開く場合には、falseを返してShadeがファイル処理を行わないようにする必要がある。

trueを返した場合、plugin_exporter_interface::get_stream_interface() または plugin_exporter_interface::get_text_stream_interface() でstream_interface を取得することができる。

falseを返した場合はstream_interface が生成されないので、ファイル保存ダイアログで指定したパスは、plugin_exporter_interface::get_file_path() で取得する。

引数:
uses_filesavedialog trueを代入すると、ファイル保存ダイアログを表示する。 変更しない、またはfalseを代入すると、ファイル保存ダイアログを省略する。

オーバライド可。uv値を伴ったポリゴンメッシュ面の情報がプラグインに渡される。

パラメータのUV値のみが渡される。UV値が不要な場合はpolymesh_face() 、距離補正のUV値が必要な場合にはpolymesh_face_uvuv() polymesh_face_uvs() を使用する。

引数:
list 面の頂点の番号の配列。例えば、面の頂点の数nが４の場合、list[0], list[1], list[2], list[3] で面を構成する頂点の番号の並びを取得できる。処理の対象となっている現在の面が頂点番号0, 5, 2, 3の並びで構成されている場合、list[0], list[1], list[2], list[3] は、それぞれ0, 5, 2, 3の値となる。
normals 法線ベクトルの配列。ポリゴンメッシュでスムーズがオンになっている場合にのみ取得できる。面の頂点の数nが4のとき、normals[0], normals[1], normals[2], normals[3] がそれぞれの頂点の法線ベクトルとなる。
