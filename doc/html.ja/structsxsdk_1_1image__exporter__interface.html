オーバライド可。複数のイメージからなるアニメーションを一個のストリームに書き出す場合には、trueを返す。

デフォルトではfalseを返し、その場合、一個のイメージのみエクスポートできることを示す。

virtual bool sxsdk::image_exporter_interface::accepts_multilayer ( int  index,
void *  aux = 0 
) [inline, virtual]

オーバライド可。複数のレイヤーを持ったイメージを一個のストリームに書き出すことができる場合には、trueを返す。

trueを返した場合、複数レイヤーを持ったイメージの保存時にdo_multilayer_export()が呼び出される。また、その呼び出しの前後にイメージのレイヤーの数だけdo_pre_multilayer_export(), do_post_multilayer_export()が呼び出される。 デフォルトではfalseを返し、その場合、一個のイメージのみエクスポートできることを示す。

引数:
index インデクス値。複数の画像形式に対応するときに種類を判別するために使用する。

オーバライド可。ファイルの種類ポップアップメニューに表示する画像ファイル形式の名称を登録する。

0を返すまでindexの値を繰り上げつつ繰り返し呼び出される。すべての画像ファイル形式を登録した後で必ず0を返す。
