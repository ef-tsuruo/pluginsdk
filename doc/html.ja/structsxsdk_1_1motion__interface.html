virtual float sxsdk::motion_interface::get_default_max_joint_value ( void *  = 0  )  const [virtual]

ジョイントのデフォルト上限値を取得する。

デフォルト上限値は、ジョイントスライダの上限の端の値を意味する。スライダの範囲はジョイントの種類によって-1.0~1.0などが決まっているが、これを変更することで-100.0~100.0などの広い範囲を扱えるスライダにすることができる。
