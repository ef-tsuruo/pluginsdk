イメージの指定したピクセルに色を設定する。(32bits/チャンネル RGB HDR)

アルファチャンネルには1.0(不透明)が設定される。アルファチャンネルを含む場合は、set_pixel_rgba_float() を使用する。

Shade 7以前では、表示用の8bits/チャンネルとは個別に、32bts/チャンネルのリアルカラーを持っていたが、Shade 8より統合されている。このため、set_pixel() と set_real_color() の描画は連動する。
