    • ラベル

  • tooltip type = xsd:string

    • ツールティップ

    • 開閉可能かどうか

  • style type = xsd:string

    • グループの枠の表示を設定します。

    設定なし: 通常の線による枠表示します。

    raised: 浮き上がったボタンの形で枠を表示します。

    depressed: 押された状態のボタンの形で枠を表示します。

  • standard type = xsd:boolean

  • professional type = xsd:boolean

  • full type = xsd:boolean
