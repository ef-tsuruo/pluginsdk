virtual void sxsdk::loader_interface::polymesh_face ( int  n_list,
const int  list[],
const sxsdk::vec3 *  normals,
const sxsdk::vec4 *  plane_equation,
const sxsdk::vec4 *  uvs,
void *  aux = 0 
) [inline, virtual]

オーバライド可。uv値を伴ったポリゴンメッシュ面の情報がプラグインに渡される。

パラメータと距離補正のUV値が渡される。UV値が不要な場合はpolymesh_face() 、パラメータのUV値のみが必要な場合にはpolymesh_face_uvuv() を使用する。

引数:
list 面の頂点の番号の配列。例えば、面の頂点の数nが４の場合、list[0], list[1], list[2], list[3] で面を構成する頂点の番号の並びを取得できる。処理の対象となっている現在の面が頂点番号0, 5, 2, 3の並びで構成されている場合、list[0], list[1], list[2], list[3] は、それぞれ0, 5, 2, 3の値となる。
normals 法線ベクトルの配列。ポリゴンメッシュでスムーズがオンになっている場合にのみ取得できる。面の頂点の数nが4のとき、normals[0], normals[1], normals[2], normals[3] がそれぞれの頂点の法線ベクトルとなる。
plane_equation 平面方程式パラメータ
uvs パラメータUV値と距離UV値の配列。vec4 の4つのデータの内、1, 2番目の値はパラメータの、3, 4番目の値は距離のUV値となる。
