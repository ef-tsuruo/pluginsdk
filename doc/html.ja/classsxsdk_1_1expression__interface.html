virtual const char* sxsdk::expression_interface::json_type_value ( void *  aux = 0  )  [virtual]

直前に実行された計算式の計算結果をJSON形式の文字列で取得する。

取得出来るのは計算された結果の変数となり、複数の計算式にエラーが含まれていた場合でも、エラーの発生する計算式を除いた式の計算結果が返される。
