XMLSHDスキーマ定義

  • joint group 8

    • ジョイント

    • type int 1

      • ジョイントの種類

      0 : ジョイント無効

      2 : 直線移動ジョイント

      3 : 拡大縮小ジョイント

      4 : 均等拡大縮小ジョイント

      5 : 光源エフェクタ

      6 : パスジョイント

      9 : 変形ジョイント

      10 : 回転ジョイント

      11 : カスタムジョイント

      12 : ボールジョイント

      13 : カメラジョイント

    • points points 1

      • ジョイントの位置等の幾何情報を表す点列

      nに続いてn個のpointが並ぶ。

      回転ジョイント：回転軸を定義する２端点

      直線移動／拡大縮小ジョイント：始点、終点

      均等拡大縮小／ボールジョイント：基準点、（表示の際の）サイズ

      • n int 1

        • 点の数

      • point vec3 1

        • 座標値

    • value float 1

      • ジョイントスカラー値

      回転ジョイント：回転角度

      直線移動ジョイント：始点-終点間の距離に対する相対値としての移動量

      均等拡大縮小／ボールジョイント：拡大縮小値

      パスジョイント：パスに対する相対値としての移動量

    • has_range int 2

      • ジョイントスカラー値が設定された範囲内に限定される

    • rotation quaternion 1

      • 回転量

      ボールジョイント、カメラジョイントで使われる

    • offset vec3 1

      • オフセット量

      ボールジョイント、カメラジョイントで使われる

    • min_value float 2

      • ユーザが設定した最小スカラー値

    • max_value float 3

      • ユーザが設定した最大スカラー値

    • default_min_value float 4

      • デフォルト最小スカラー値

      ジョイントの種類によって異なった値がデフォルトとなることに注意

    • default_max_value float 5

      • デフォルト最大スカラー値

      ジョイントの種類によって異なった値がデフォルトとなることに注意

    • default_scalar float 6

      • デフォルトスカラー値

      ジョイントの種類によって異なった値がデフォルトとなることに注意

    • blend float 7

      • ブレンド値

      スキンをジョイントにバインドする際のブレンド値

    • default_rotation quaternion 2

      • デフォルト回転値

    • transformation mat4 1

      • 変換マトリクス

      ジョイントの評価結果としての変換マトリクス値。Shade本体では使用しないが、ジョイント機能を実装しないプログラムで読み込む際には、実際にジョイントを評価して変換マトリクスを計算する必要はなく、このマトリクス値を使えば良い。

    • graph_height int 3

      • モーショングラフを画面に表示する際の、グラフの高さ

    • flags

      • 各種フラグ

      • flat_joint 0x1

        • フラットジョイント

        ONの場合には、ジョイントパートに含まれる子形状ではなく、同階層でそれ以降にならぶ形状に対してジョイントが作用する。

        デフォルトでOFF。

      • path_correction_distance 0x20

        • パス補正(パスジョイントのみで有効)

        ONの場合には、パスの線形状を距離で補正して作用する。

        Shade 8で作成したパスジョイントはデフォルトでON。Shade 7以前で作成したファイルを開いた場合はOFF。

      • graph_disclosed 0x10

        • モーション設定グラフをディスクローズ表示

      • tree_disclosed 0x8

        • モーション設定ツリーをディスクローズ表示

  • shape_specifier group 133

    • 形状を特定するために必要な情報

    リンクの種類の後に１個または複数個並んだ整数値によって、形状を特定する。整数値は、ルートレベルから辿って、階層内での対象形状またはそれを含むパートの相対位置を表す。したがって、必ず 0 から始まり、[0]はルート形状を指す。

    • link_type int 1

      • リンクの種類

      0 : 通常のリンク形状

      1 : カメラの視点連結形状

      2 : カメラの注視点連結形状

    • int

      • 階層内での相対位置

  • attribute group 103

    • プラグインが付加したカスタム属性データ

    • uuid uuid 1

      • プラグインがカスタム属性データを特定するための識別子

    • wireframe_uuid uuid 2

      • ワイヤフレーム表示を行うプラグインを特定する識別子

    • intersection_uuid uuid 3

      • レイトレーシング交点計算を行うプラグインを特定する識別子

    • stream block 1

      • データを保持するストリーム

      • size

        • ストリームのサイズ（バイト数）

    • label string 1

      • プラグインから自由に設定、取得できる名称

  • attributes group 102

    • プラグインが付加したカスタム属性の配列

    （ポリゴンメッシュの頂点に割り当てられたattributesの場合には面インデクス値、頂点インデクスに続き）0個またはそれ以上のカスタム属性が並ぶ。

    • face_index int 1

      • 面インデクス値

      ポリゴンメッシュの面の頂点に割り当てられたカスタム属性の場合には、割り当てられた面のインデクス値を表す。それ以外の場合には無効。

    • vertex_index int 2

      • 頂点インデクス

      ポリゴンメッシュの面の頂点に割り当てられたカスタム属性の場合には、割り当てられた頂点の（面内）インデクス値を表す。それ以外の場合には無効。

    • attribute group 103

      • attributesに含まれる個々のカスタム属性の実体

      -1 : 親の値を継承

    • radiosity_doublesided int 12

        • 頂点

          • 座標値の配列

          座標値の数に続き、座標値のX,Y,Zが繰り返し並ぶ。

        • active_flags ints 1

          • アクティブかどうかを示すフラグの列

          フラグの数に続き、0または1のフラグ値が繰り返し並ぶ。

        • skin group 110

          • スキン設定

          0個以上のスキン設定がverticesグループ内に含まれる。

          個々のスキン設定がどの頂点に割り当てられているかは、skinグループ内のインデクス値で判断する。

      • edges group 114

        • 稜線

        • edge_vertices ints 1

          • 稜線を成す頂点インデクス値ペアの列

          頂点インデクス値の数（稜線の数の２倍）に続き、稜線の両端点の頂点インデクス値が並ぶ。

      • faces group 115

        • 面

        • face_vertices ints 1

          • 面を成す頂点インデクス値の列

          先頭の値は、それ以降に続く値の数を示し、続いて個々の面ごとにその面の頂点数と頂点インデクス値の列が繰り返し並ぶ。

        • face_uvs floats 1

          • 面を成す頂点に割り当てられたUV値の列

          先頭の値は、それ以降に続く値の数を示し、続いて個々の面ごとその個々の頂点に割り当てられているU_PARAMETER、V_PARAMETER、U_DISTANCE、V_DISTANCEの値が繰り返し並ぶ。

        • index int 1

          • 面インデクス

        • normals vecs 1

          • 法線ベクトルの配列

          一個の面に属する全ての頂点の法線ベクトルの配列。先頭の値はその面が持つ法線ベクトル、すなわち頂点の数。それに続いて法線ベクトルのX,Y,Z値が並ぶ。

          法線ベクトルがどの面に割り当てられているかは、直近に読み込まれた面インデクスの値で判断する。

        • subsurface_index int 2

          • サブ面インデクス

          一個のポリゴンメッシュ内の個々の面に異なった表面材質を割り当てる場合の、表面材質インデクス値。

          サブ面インデクスがどの面に割り当てられているかは、直近に読み込まれた面インデクスの値で判断する。
