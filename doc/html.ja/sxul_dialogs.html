  • id type = xsd:integer

    • 識別子

  • swap_boxes type = xsd:boolean

    • trueを指定すると、内包する複数のbox itemを同じ場所に重ね、そのうち一つを表示するようになります。

    tab itemと同じ記述になります。

    表示上は通常のグループと同じになり、選択されているコントロールのセットのlabelがタイトルになります。

    コントロールセットの切り替え、取得はset_selection(), get_selection()で行います。

  • label type = xsd:string

    • ラベル

push-button

プッシュボタン

  • id type = xsd:integer

    • 識別子

  • label type = xsd:string

    • ラベル

  • width type = xsd:integer

    • 幅

    指定したサイズに比べて文字や画像の幅が大きい場合は、コントロールの幅を自動的に広げます。

  • height type = xsd:integer

    • 高さ

    指定したサイズに比べて文字や画像の幅が大きい場合は、コントロールの幅を自動的に広げます。

  • image type = xsd:string

    • targa画像リソースから画像つきのボタンを作成

    ボタンの状態に対する描画状態はOSの標準に合わせています。(画像にアルファチャンネルを設定することを推奨)

  • alternate_image type = xsd:string

    • ボタン押下時に表示される画像を設定

    画像を変更しない場合は指定する必要はありません。

  • tooltip type = xsd:string

    • ツールティップ

  • terminator type = xsd:boolean

    • trueの場合、このプッシュボタンをクリックすると、ダイアログが閉じられる。

  • id type = xsd:integer

    • 識別子

  • label type = xsd:string

    • ラベル

  • push type = xsd:boolean

    • チェックボックスをボタン型に設定

    imageを設定した場合は自動的にボタン型になります。

    • チェックボックスの幅

    指定したサイズに比べて文字や画像の幅が大きい場合は、コントロールの幅を自動的に広げます。

  • height type = xsd:integer

    • チェックボックスの高さ

    指定したサイズに比べて文字や画像の幅が大きい場合は、コントロールの幅を自動的に広げます。

  • image type = xsd:string

    • targa画像リソースから画像つきのボタンを作成

    ボタンの状態に対する描画状態はOSの標準に合わせています。(画像にアルファチャンネルを設定することを推奨)

  • alternate_image type = xsd:string

    • ボタン押下時に表示される画像を設定

    画像を変更しない場合は指定する必要はありません。

  • bordered type = xsd:boolean

    • 枠の描画

    falseを指定すると枠なしのボタンになります。

    画像を貼り付けたボタンのみ有効です。

    テキストが編集されると、respond()の呼び出しでactionにsx::dialog_item::item_changedが渡されます。

custom

カスタムアイテム

縦、横のサイズを固定する場合は、最大サイズと最小サイズを初期値と同じ値に設定します。

動作中にサイズを変更した場合は、window_interface::resize() が呼ばれます。

オーバーライドする場合は、関数内で必ず、window_interface::resize() を呼び出す必要があります。

サイズの変更に合わせて再描画する場合は、window_interface::paint() を呼び出します。
