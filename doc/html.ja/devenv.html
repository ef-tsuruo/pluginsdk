開発環境

■プラグインの開発環境

Shade Plugin SDKによるプラグイン開発言語はC++です。プラグインを開発するには以下に示す開発環境が別途必要になります。

Windows版

OS

  • Windows Vista 32bit / 64bit
  • Windows XP Professional x64 Edition
  • Windows XP Professional / Home Edition / Media Center Edition (SP2以降)

開発ツール

  • Visual Studio 2005 Team System / Professional Edition / Standard Edition
  • Visual C++ 2005 Express Edition

※ Windows 95 / Me / NT / 2000 には対応していません。
※ Visual Studio .NET 2003、Visual C++ .NET 2003、Visual Studio .NET 2002、Visual C++ .NET 2002、Visual C++ 6.0には対応していません。
※ Visual Studio 2005の利用言語は、Visual C++ 2005となります。
※ Visual C++ 2005 Express Editionでは、64bit版のプラグインは開発できません。
※ Visual Studio 2005およびVisual C++ 2005利用時は、Visual Studio 2005 Service Pack 1の適用を推奨します。

　 Visual Studio 2005 Service Pack 1
　 http://www.microsoft.com/japan/msdn/vstudio/downloads/sp/vs2005/sp1/

Mac OS X版

OS

  • Mac OS X 10.4.7以降

開発ツール

  • Xcode 2.4以降

※ Classic環境あるいはMac OS 9、MacOSX10.3以前には対応しておりません。
※ CodeWarrior、Project Bulider、Xcode2.3以前には対応しておりません。
※ Xcodeは最新のバージョンを利用することを推奨します。

　 Apple Developer Connection
　 http://developer.apple.com/jp/

■Visual C++ 2005 Express Editionでのプラグイン開発について

Visual C++ 2005 Express EditionでShade 10 プラグインの開発を行うには、Microsoft Platform SDKが必要です。
Visual Studio 2005 Standard Edition以上では、Platform SDKは組み込まれていますので設定は不要です。

・ Visual C++ 2005 Express EditionおよびMicrosoft Platform SDKを以下のWebページからプログラムをダウンロードし、インストールします。

　 Visual C++ 2005 Express Edition
　 http://www.microsoft.com/japan/msdn/vstudio/express/visualc/

　 Microsoft Platform SDK (「PSDK-x86.exe」をダウンロード)
　 http://www.microsoft.com/downloads/details.aspx?FamilyId=A55B6B43-E24F-4EA3-A93E-40C0EC4F68E5

・ インストールが完了したら下記ページの説明に従ってVisual C++ 2005 Express Editionの設定を行ってください。

　 Visual C++ 2005 Express Edition と Microsoft Platform SDK を一緒に使う
　 http://www.microsoft.com/japan/msdn/vstudio/express/visualc/usingpsdk/

設定が完了するとShade 10 プラグインの開発が行えるようになります。

■プラグインの動作環境

Shade 10 Plugin SDKで作成したプラグインはShade 10 Professional/Standardで動作します。Basicではプラグインがサポートされていないため動作しません。
